# Печатайте любые символы напрямую с клавиатуры в стиле hypetype

Делал hypetype для себя так как страдал от ограниченности ввода специальных символов.
Пригодится дизайнерам, редакторам, корректорам, техиписам, программистам и прочим любителям типографики.

Полное описание, инструкции по установке и ответы на вопросы есть в репозиториях
